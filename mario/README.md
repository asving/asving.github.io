# Basic Mario Clone

A simple Mario-style platformer game built with pygame.

## Requirements

```bash
pip install pygame
```

## How to Run

```bash
python mario.py
```

## Controls

- **Arrow keys** or **WASD**: Move left/right
- **Space** or **W** or **Up arrow**: Jump
- **Q**: Quit

## Features

- Player character (Mario)
- Platforms and bricks
- Question blocks
- Collectible coins (+100 points each)
- Goombas (enemies) - jump on them to defeat (+200 points)
- Pipes (obstacles)
- Gravity and collision physics
- Score tracking

## Gameplay

- Collect coins for points
- Jump on Goombas to defeat them
- Avoid touching Goombas from the side (resets position)
- Don't fall off the screen!
